# Приют и Ветклиника Пушистый друг

Современный веб-сервис для управления ветеринарной клиникой и приютом для животных. Проект представляет собой полноценную систему с динамическим контентом, управляемым через удобную административную панель.

Проект мы делали в команде с **Алексеем Прокороновым**. Мы чётко разделили зоны ответственности:
* Алексей полностью взял на себя фронтенд и визуальную часть сайта (HTML, CSS, JS). Он сделал так, чтобы сайт выглядел красиво, современно и адаптивно.
* Я (Бэкенд-разработчик) отвечал за весь бэкенд и "внутренности" проекта. На мне была архитектура базы данных, настройка моделей, написание логики для вычислений и полная настройка админ-панели Django, через которую сейчас управляется весь контент сайта. Я сделал так, чтобы сайт реально работал и обрабатывал данные.

## Что использовалось???
* Бэкенд: Python, Django 6.0
* *База данных: SQLite
* Фронтенд: HTML5, CSS3, Vanilla JS
* **Медиа: Pillow (работа с изображениями)

## Основной функционал
* Сайт-визитка: Динамический вывод актуальных питомцев, персонала и дневника лечения на главную страницу.
* Админ-панель: Полный контроль над контентом сайта. Возможность добавлять новых животных, врачей, указывать породы, прикреплять фотографии и назначать лечение.
* Автоматизация: Умная сортировка (самые новые животные отображаются первыми), автоматический подсчёт статистики по свободным и пристроенным животным.
